🌍 PlanetPal

Assignment 3 - Progressive Web App

About the App

PlanetPal is an educational Progressive Web App designed to help users learn about the Solar System. It provides interactive topics, images, and audio explanations for each planet.

Features

  • Dynamic menu powered by JSON data
  • Images and audio for each topic
  • Works offline using Service Worker
  • Installable as a mobile/desktop app

App Preview

How to Install

Open the web app and click the Install App button, or use your browser’s menu and select “Add to Home Screen”.
