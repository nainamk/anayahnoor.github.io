PlanetPal Instructions

User Instructions

Open the app and choose a planet from the topic menu. Read the information and play the audio.

Installation

Click the Install App button or use “Add to Home Screen” from your browser menu.

Programmer Documentation

To add a new topic, open data.json and add a new object inside the planets array. 

{
  "title": "Jupiter",
  "description": "Jupiter is the largest planet.",
  "image": "image_url_here",
  "audio": "audio_file_here"
}
